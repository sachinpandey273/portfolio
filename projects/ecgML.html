Real-time Heart Rate Estimation via DeepPhys

  • Tech Stack: Python, Deep Learning, OpenCV, PyTorch, rPPG
  • Github URL: Project Link

Overview

A deep-learning–based system for real-time, non-contact heart rate estimation using remote photoplethysmography (rPPG). This project implements the DeepPhys model, leveraging convolutional neural networks to extract subtle color variations from facial video frames and convert them into accurate pulse signals.

Features

  • Non-Contact Measurement: Estimate heart rate continuously without physical sensors.
  • DeepPhys Model: Utilizes an end-to-end convolutional neural network for remote photoplethysmography (rPPG).
  • Real-Time Video Analysis: Extracts and processes subtle skin color variations from facial video frames in real-time.
  • Robust Feature Extraction: Effectively handles lighting conditions and motion artifacts to ensure accurate physiological signals.
  • Interactive UI: Clear visualization of the extracted pulse signals and estimated heart rate.

Technologies Used

  • Deep Learning: PyTorch, Convolutional Neural Networks (CNNs)
  • Computer Vision: OpenCV for robust face detection and video frame processing
  • Programming: Python
  • Signal Processing: SciPy and NumPy for extracting and validating pulse metrics
